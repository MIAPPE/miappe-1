Support

General questions regarding MIAPPE usage can be asked on the MIAPPE mailing list or through miappe-feedback@ebi.ac.uk.

Documentations and Resources

Specification
  • The current version of MIAPPE (released 10th January 2019), MIAPPE v1.1 (PDF format .xlsx format) with training material and data templates .
  • The previous versions of MIAPPE can be found on our github
Publications
See the Publication page for detailled description of the principles, the history and the impolementation of MIAPPE.
Implementations and Tools
  • The Breeding API is a web service specificaiton that implements several standards including MIAPPE
  • Recomended templates can be used for data exchange, database submissions, and training.
  • The current implementation of the MIAPPE standard in ISA-Tab format.
Miscellaneous links
  • A full list of Minimum Information Standards available, and the MIAPPE record, available on biosharing.org.
  • More information about the ISA-tab for phenotyping: concept and use; and more generally, about the Investigation-Study-Assay file format.
  • A mapping of MIAPPE terms to the checklist for the submission of plant nucleic acid sequence to the European Nucleotide Archive (coming soon).
Project resources
  • The MIAPPE GitHub repository.
  • The associated GitHub issue tracker.
  • An archive of previous versions of the MIAPPE specification and ISA-Tab configurations.
